## Agents

- MVP Program Manager: Coordinates the Latvian A2 Exam Simulator MVP, decomposes work into issues, tracks dependencies, and keeps release scope aligned to commercial MVP.
- Product UX Exam Flow Agent: Builds the learner-facing exam flow, practice/real modes, hidden debug panels, result screens, and polished UX.
- Backend Persistence Agent: Builds the FastAPI/Postgres backend for users, exams, attempts, answers, scores, AI evaluations, payments, and history.
- Auth Accounts Agent: Implements auth, profiles, dashboard, protected routes, account recovery, deletion/export, and auth webhook sync.
- Payments Entitlements Agent: Implements Stripe products, Checkout, billing page, entitlements, paid exam access, credits, and webhooks.
- AI Scoring Reliability Agent: Implements writing/speaking AI scoring with rubrics, retries, quotas, failure states, cost controls, and audit logs.
- Content Bank QA Agent: Converts Markdown exams to structured JSON, validates schema/assets/scoring, and creates content governance workflow.
- DevOps Security Ops Agent: Deploys and operates the paid product with HTTPS, secrets, monitoring, backups, rate limits, and legal pages.
- QA Release Agent: Owns regression tests, E2E flows, CI readiness, release checklist, and production acceptance validation.
